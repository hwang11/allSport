# allSport
### 안드로이드 & ios 앱
대한체육회 대회 정보를 확인하고 관람 모임을 구할 수 있는 앱입니다.   
####1. 기술스택  
* Main : Java, Spring-boot, React-native, MySQL  
* Module : Apache POI, STOMP

####2. 기능  
* 대한체육회 대회 정보 확인  
* 대회 후기 작성  
* 대회 관람 모임 조성  
* 채팅  

#### 어떻게 실행하나요 
~~~
$gradle bootrun 
~~~
